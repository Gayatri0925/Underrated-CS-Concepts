## Day 12 – Bulkhead Pattern

The Bulkhead Pattern isolates system components to prevent cascading failures.

### How it works
- Separate resources into isolated pools
- Failure in one does not affect others
- Critical services remain available

### Why it matters
- Improves resilience
- Prevents cascading failures
- Better resource utilization
